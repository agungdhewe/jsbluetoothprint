Mencetak ke Printer Bluetooth menggunakan library POS Encoder dari Web Browser

Untuk mempermudah dan pengembangan, kita bisa menggunakan library esc-pos-encoder, yang bisa diambil dari https://github.com/NielsLeenheer/EscPosEncoder/blob/master/dist/esc-pos-encoder.esm.js

atau dari npm di https://www.npmjs.com/package/esc-pos-encoder

Untuk menggunakan library ini, pada modul lakukan import sbb:

import EscPosEncoder from './esc-pos-encoder.esm.js'

const posEncoder = new EscPosEncoder({
    width: 36 //lebar kertas
})
